DIETETIC ADVICE SECOND MEETING ASSISTANT

TO RUN:
1 - run the single runnable containing both python files from a command prompt using:
	python dieteticsystem.py
2 - import the folder into a python project environment such a PyCHarm and run system.py through the software.

FUNCTIONALITY:
The system will ask inputs from the user regarding a gastric surgery patient's second meeting since the surgery.
The values to be entered will be asked for one by one, after which an overview is presented,
 in which advice is proposed to the dietitian.

OUTPUT:
The system stores the overview of the meeting in patientname_overview.txt
